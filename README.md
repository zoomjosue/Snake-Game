# Snake Game

Juego clasico de Snake con pantalla de inicio, niveles de dificultad, puntaje y record.

## Ejecutar

```bash
npm install
npm run dev
```

Abre `http://localhost:5173` en tu navegador.

## Como jugar

1. Selecciona la dificultad: Facil, Normal o Dificil.
2. Presiona JUGAR.
3. Usa las flechas o `W A S D` para mover la serpiente.
4. Come la comida para crecer y sumar puntos.
5. Evita chocar con las paredes y con tu propio cuerpo.

El juego aumenta la velocidad cada 50 puntos.
